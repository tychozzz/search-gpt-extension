## 简介

Search GPT 是一个浏览器插件，它将 ChatGPT 的能力赋予浏览器。当安装插件并在浏览器（谷歌或百度）中搜索某些内容时，页面中同时会展示 GPT 响应。

插件基于 Plasmo 与 React 实现。

## 功能

- ✨ 在浏览器页面展示 GPT 响应内容。暂时支持 Google 与 Baidu。

- 🌟 总结每个搜索结果页内容。

## 快速开始

1.   初始化

     ```bash
     pnpm install
     ```

2.   运行并构建打包插件

     ```bash
     pnpm run dev
     ```

3.   以插件形式加载已构建好的 `build/chrome-mv3-dev` 文件夹
